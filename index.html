Let's brick

Getting started is easy.

If you want the entire front-end structure:

              
# Clone this repository
$ git clone https://github.com/Riccardo-Zanutta/brick.git
$ cd your_next_awesome_project

# Install dependencies
$ npm install
# Run tasks and serve
$ npm run start
# or
$ gulp --pretty

If you want only the css library:

                
# Install with npm
npm install brick-css

# Install with bower
bower install brick-css
